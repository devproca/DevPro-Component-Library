Getting Started

The checkbox component can be used as a single basic checkbox or grouped together with the CheckBoxGroupComponent component for multi-selection.

Import

Currently everything is imported by DevproComponentsModule.

Label

Labels can be added to the checkbox as an attribute. Clicking on the label will also toggle the checkboxes checked state.

{{ codeLabel }}

Value and Forms

Value can be assigned to each checkbox as an attribute. Adding a FormControlDirective to the CheckboxGroupComponent or and individual CheckboxComponent will allow you to utilize the information each checkbox is intended to provide.

{{ codeValue }}

Properties

Name Type Default Value Description
label string null Label for the checkbox.
value any null Value of the checkbox.
disabled boolean false Disabled attribute of the checkbox, when true it will visibly change the checkbox and make it un-clickable.

Events

Name Parameters Description
OnClick isChecked: Boolean value set by event emitters dictating the state of the checkbox. Notifies changes to the CheckboxService on the checked status.

Styling

Name Element
checkbox-wrapper Outermost element containing the input and label.
checkbox-container Container element for the custom checkbox input.
checkbox-svg SVG Element for the 'check' in the checkbox.
