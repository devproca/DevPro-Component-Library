Alert

Custom messages used to alert the user.

Clear All

Getting Started

Alerts can be generated through the AlertService and displayed in the AlertsComponent, multiple alerts can be generated in a single element. The AlertComponent can also be manually created and assigned its type, message, and details attribute.

Import

Currently everything is imported by DevproComponentsModule.

Message, Details, and Type

The alerts message and details can be added to the alert easily through a service call. The options for type include: Success, Info, Warning, and Danger. Type is dictated by which function is called in the service.

Properties

Name Type Default Value Description
type string success The type of the alert dictating its appearance.
message string null The message displayed in the alert.
details string null Optional details under the message to provide more information about the alert.

Events

Name Parameters Description
onClose close: Event emitter for the close button. Clears the alert element.

Styling

Name Element
