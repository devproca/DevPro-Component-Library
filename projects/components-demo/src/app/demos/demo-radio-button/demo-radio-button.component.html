Getting Started

The radio button component does not function on its own and must be grouped together with the RadioGroupComponent so that only one button can be selected at a time.

{{ codeGroup }}

Import

Currently everything is imported by DevproComponentsModule.

Label

Labels can be added to the radio button as an attribute. Clicking on the label will also toggle the buttons checked state.

{{ codeLabel }}

Value and Forms

Value can be assigned to each button as an attribute. Adding a FormControlDirective to the RadioGroupComponent will allow you to utilize the information each button is intended to provide.

{{ codeValue }}

Properties

Name Type Default Value Description
label string null Label for the radio button.
value any null Value of the radio button.
disabled boolean false Disabled attribute of the radio button, when true it will visibly change the radio button and make it un-clickable.

Events

Name Parameters Description
OnChecked isChecked: Boolean value set by event emitters dictating the state of the radio button. Notifies changes to the RadioService on the checked status.

Styling

Name Element
radio-button-wrapper Outermost element containing the custom button and its label.
radio-button Overlays the hidden input and acts as the visual button.
radio-button-label Label element for the radio button.
