Getting Started

The button component can be used in many applications. It will act like an HTML <button> element and accept click() attributes. The element comes with a default configuration of basic and flat.

Import

Currently everything is imported by DevproComponentsModule.

Properties

Name Type Default Value Description
color - basic Dictates the color of the button.
stroked boolean false Dictates if the button is solid or stroked.
raised boolean false Dictates if the button is flat or raised.
disabled boolean false Disabled attribute of the button, when true it will visibly change the button and make it un-clickable.

Styling

Name Element
button-wrapper Outermost element containing the styling and the content within.

Color

The color can be changed with the color attribute. Options include: basic, primary, and secondary. The default is basic.

Stroked

The stroke styling can be enabled by setting the stroked attribute to true. The default is false.

Raised

The raised styling can be enabled by setting the raised attribute to true. The default is false.

Disabled

The disabled styling can be enabled by setting the disabled attribute to true. The default is false.
