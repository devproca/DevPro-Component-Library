Overflow Menu

A floating menu panel that contains a given list of options or an ng-template.

Basic Options Menu

{{ option.label }}

Getting Started

The overflow menu component requires a list of options to create an OverflowMenuModel with. These options can consist of a simple label and click() method or multiple attributes that can be organized inside of the optionally provided ng-template element.

Import

Currently everything is imported by DevproComponentsModule.

Options

An array of options can be provided to the component through its options attribute. These options at a minimum should provide a label to display in the menu and something to be done on clicking the option.

Templated Options

When using a templated menu, these options can be expanded to include additional details for each option that can be utilized in your ng-template element.

Properties

Name Type Default Value Description
options OverflowMenuModel null The model used for creating the component.
template TemplateRef<any> null An ng-template reference to customize the overflow menus appearance.

Styling

Name Element
menu-wrapper Outermost element containing the menu button and item list.
item-wrapper Container element for the popup menu.
item Styling for each menu option.

Methods

None.

Dependencies

The menu utilizes a PopperComponent that requires the Popper.js dependency.
