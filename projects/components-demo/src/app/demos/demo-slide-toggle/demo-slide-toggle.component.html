Getting Started

The slide toggle component can be used as a single basic toggle or grouped together with the SlideGroupComponent component for multi-selection.

Import

Currently everything is imported by DevproComponentsModule.

Label

Labels can be added to the slide toggle as an attribute. Clicking on the label will also toggle the slide on state.

Value and Forms

Value can be assigned to each toggle as an attribute. Adding a FormControlDirective to the SlideGroupComponent or and individual SlideToggleComponent will allow you to utilize the information each button is intended to provide.

Properties

Name Type Default Value Description
label string null Label for the slide toggle.
value any null Value of the slide toggle.
disabled boolean false Disabled attribute of the slide toggle, when true it will visibly change the toggle and make it un-clickable.
alternateStyling boolean false Boolean attribute that enables or disables the alternate styling option for the slide toggle.

Events

Name Parameters Description
OnClick isChecked: Boolean value set by event emitters dictating the state of the slide toggle. Notifies changes to the SlideToggleService on the checked status.

Styling

Name Element
slide-toggle Outermost element containing the input, custom paths/buttons, and label.
styling Container element for the universal styles for both options.
toggle-path The path the button travels across.
toggle-button The round button that travels across the path when sliding.
toggle-label Label element for the slide toggle.

Alternate Toggle Style

The default styling for the slide toggle is more self contained and suited for stacking several together. If you prefer the asymmetrical larger styling it can be enabled by setting the alternateStyling attribute to 'true'.
