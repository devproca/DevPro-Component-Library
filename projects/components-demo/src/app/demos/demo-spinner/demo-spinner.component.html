Getting Started

The spinner component can be used in many applications. The element comes with a default configuration of blue coloring, style 1, and size extra small. But can be configured in several different ways.

Import

Currently everything is imported by DevproComponentsModule.

Properties

Name Type Default Value Description
size - xs Dictates how large the spinner will be.
spinnerStyle - style1 Dictates the style of the spinner.
color - blue Dictates the color of the spinner.

Styling

Name Element
spinner-wrapper Outermost element containing the animation, color, style, and size.

Style

The style can be changed with the spinnerStyle attribute. Options include: xs, sm, md, and lg. The default is style1.

Size

The size can be changed with the size attribute. Options include: style1, style2, and style3'. The default is extra small.

Color

The color can be changed with the color attribute. Options include: blue and white. The default is blue.
