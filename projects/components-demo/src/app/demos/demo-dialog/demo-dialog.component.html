Dialog

A modal component that displays a child window in front of the main window.

Open Dialog Example

Getting Started

A dialog component can consist of DialogHeaderComponent, DialogBodyComponent, and a DialogFooterComponent like the above example, however other components can be injected.

Import

Currently everything is imported by DevproComponentsModule.

Dialog Service

Dialog modals can be opened through the DialogService by calling the open function.

Dialog Config

The DialogService open function can take an optional DialogConfig with a data and size parameter.

Properties

Name Type Default Value Description
size string null Variable determining the size of the modal.
state string out State for the overlays fading animation, in or out.

Styling

Name Element
overlay Outermost element that fades out the main window behind the modal.
dialog Container element for the modals basic styling and size.

Methods

Name Parameters Details
close result: any (Optional) Closes the modal and returns to the main window.

Service Methods

Name Parameters Details
open
  • componentType: any
  • DialogConfig (Optional)
    		•  Opens the given component in a dialog window.
    destroyCallback - Destroys the dialog component reference.

    Dependencies

    None.

    The example consists of a random component being fed into a DialogComponent. The RandomComponent is made up of a DialogHeaderComponent, DialogBodyComponent, and a DialogFooterComponent which are all included in the library.
