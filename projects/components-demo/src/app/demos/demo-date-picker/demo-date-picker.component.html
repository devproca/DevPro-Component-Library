Properties

Name Type Default Value Description
placeholder string null Placeholder for the input field.
value any null Value of the date picker.
disabled boolean false Disabled attribute of the date picker, when true it will visibly change the input field and make it un-clickable.
minDate - - Minimum date allowed to be input/selected.
maxDate - - Maximum date allowed to be input/selected.
