Side Navigation Bar

A side bar component that displays a child window in front of the main window for navigating around the application.

Open Side Nav

Getting Started

A side nav component requires a SidenavHeaderComponent, SidenavContentComponent, and a SideNavComponent.

Import

Currently everything is imported by DevproComponentsModule.

Header

The SidenavHeaderComponent is a button that you would add into your application in an accessible location such as the navigation bar, header, etc. The button can be sized and styled and clicking it will open the side bar.

{{ codeHeader }}

Content

The SideNavContentComponent is contained in a ng-template and is the list of navigation items populating the body of the side bar. These items should be defined in a typescript file with a label, route, and optionally a fragment to navigate too when clicked. These items should match any router paths set in your app-routing.module.ts file to function properly.

{{ codeContent }} {{ codeContentTS }}

SideNav

The SideNavComponent is the body of the side bar and sets its styling details for template reference, width, title, animation duration, and open direction. These details can be set in the elements attributes.

{{ codeBody }}

Properties

Name Type Default Value Description
sideNavRef any null References the tag for the ng-template containing the SidenavContentComponent.
sideNavWidth number window.innerWidth Defines the width of the side bar in pixels.
sideNavTitle string null Defines the title at the top of the side bar.
openDirection SideNavDirection SideNavDirection.Left Defines if the side bar should open from the left or the right.
duration number 0.25 Defines the animations time to open the side bar in seconds.

Styling

SideNav

Name Element
sidenav-wrapper Outermost element that contains the overlay, menu, direction, etc for the side bar.
sidenav-overlay Element that fades out the main window behind the window.
sidenav-menu Container element that contains the styling for the menu defined in its attributes.
sidenav-header Container element for the header portion of the side bar.
sidenav-content Container element for the template reference content given to the sidenav.
