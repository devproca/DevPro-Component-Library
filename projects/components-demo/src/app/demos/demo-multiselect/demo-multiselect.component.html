Multiselect Input

A selectable input component for choosing multiple values from a drop down menu. Utilizes the same component as the SelectComponent. Supports ControlValueAccessor.

Getting Started

To enable multi-select in the SelectComponent, add the [multiselect]="true" attribute. The multi-select component can be used with Angular Form Control.

Import

Currently everything is imported by DevproComponentsModule.

Options

The values contained the SelectComponent's drop down can be easily passed in with the options attribute. The options can be defined in a typescript file.

Properties

Name Type Default Value Description
placeholder string null Placeholder for the input field.
multiselect boolean false Must be set to true to enable multi-select.
error boolean false Error state for error styling.
disabled boolean false Disabled attribute of the select component, when true it will visibly change the input field and make it un-clickable.

Events

Name Parameters Description
notifyChanges change: Emitted string that is a new change in selected value(s). Notifies changes to the state of the field and its value(s).

Styling

Name Element
select Outer element containing the input box and its attributes.
selected-item-wrapper Container element for the displaying the selected values.
filter Styling for the filtering done on the options when searching.
item-wrapper Outer element containing the items in the input field and the check beside already selected items.
item Element displaying the options in the drop down.

Methods

None.

Dependencies

The dropdown utilizes a PopperComponent that requires the Popper.js dependency.
