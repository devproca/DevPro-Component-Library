Chips

Simple component for displaying multiple values, keyboard accessible.

Getting Started

The chip component can be used as a single element or within another element such as a text-box.

Import

Currently everything is imported by DevproComponentsModule.

Label

Labels can be added to the chip as an attribute.

{{ codeLabel }}

Properties

Name Type Default Value Description
label string null Label for the chip.
disabled boolean false Disabled attribute of the chip, when true will visibly change the chip and make it un-clickable.

Events

Name Parameters Description
onClose close: Event emitter for the close button. Clears the chip element.

Styling

Name Element
chip-wrapper Outermost element containing the label and icon.
chip-label Label element for the chip.

None.
