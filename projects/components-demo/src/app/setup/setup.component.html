Getting Started

Download

DevPro components are available through NPM, once your application is setup run the following commands to install the library into your project: npm install devpro-components --save

Import

All components are currently exported by the DevproComponentsModule and can be imported by adding the following line in your app.module.ts file. import {{"{"}} DevproComponentsModule } from 'devpro-components';
